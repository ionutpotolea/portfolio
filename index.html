A selection of my range of work

TechCube

This is the capstone project for the JavaScript course I followed at The informal School of IT.

The project uses a Firebase Realtime Database to store the product information. It is a CRUD app that also uses localstorage to store the cart.

The code is organized following the MVC - Model-View-Controller Pattern. The project uses modern ES6 JavaScript concepts such as classes, async/await, let/const. Also regex validations were used for form input fields. The app keeps track of the stock of products. If a user orders a product, that product is automatically removed from the stock count. If a product becomes out of stock, the users have the option to setup an email alert when the product becomes available.

Languages used: HTML, CSS, JavaScript
